El proyecto en su conjunto esta basado en la fabrica en la que trabajo.
Los productos, fotos y precios son reales. La utilidad seria inmediata.
El unico aspecto a modificar para la version definitiva, y fue agregado solo por cuestiones de
rubricas de evaluación, es el stock, ya que la fabrica no trabaja con stock, si no que fabrica contra pedido.

En cuanto a algunas desiciones de diseño:

 - Incluí SweetAlert para "adornar" la experiencia del usuario.
 - Los estilos en general estan aplicados con Bootstrap.

El enfoque desde el principio fue crear una web que no sea solo un proyecto educativo, si no mas bien
una web que pueda tener una futura utilidad, tanto para esta fabrica como para cualquier otra.

Concluyo diciendo que hace dos meses me resultaba completamente impensado llevar a cabo un proyecto asi.
